# flutter-sayfalar-arasi-gecis
### Uygulama Senaryosu (!)
- flutter bootcamp ödevlerinden bir olan bu basit program içerisinde : pop , push , pushReplacement ve WillPopScop işlevleri barındırmaktadır.
